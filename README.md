### Projects

**Union AI — Agentic AI Mobile App** (React Native)
Built the message rendering system for the app's chat UI — 20+ message types across 16 custom UI renderers, covering user interactions, AI responses, event forms, tool-use results, and streaming AI output displayed progressively mid-conversation. Also built the event creation flow (describe an event in chat, the agent builds it, sends a confirmation form, then handles invites — all inside the chat UI), plus animations for long-press actions, reactions, image preview, and an interactive Google Maps location picker. 520+ contributions in 2025.

**[Tailor-Tap](https://github.com/shehzar-abbasi-SG/Tailor-Tap)**
React Native mobile app built for a local client — a tailor.

**[AMG_MVP](https://github.com/shehzar-abbasi-SG/AMG_MVP)**
Experimentation project rebuilding an existing Magento site's frontend on Next.js with Shopify as a headless CMS. Proved out the Shopify headless integration; the client ultimately chose to stay fully on Shopify instead.
